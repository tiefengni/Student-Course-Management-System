# Student Course Management System

A Java console-based application for managing students, courses, enrollments, and grades.

## Description

This project simulates a basic course management system. It allows users to add students, add courses, enroll students in courses, assign grades, and display student/course information.

The goal of this project is to practice object-oriented programming, Java collections, and basic software design.

## Features

- Add students
- Add courses
- Enroll students in courses
- Assign grades to students
- Display student information
- Display course information

## Technologies Used

- Java
- Object-Oriented Programming
- ArrayList
- HashMap
- Scanner

## What I Learned

Through this project, I practiced designing Java classes, using object-oriented programming principles, managing data with Java collections, and building a simple console-based application.

## How to Run

1. Clone the repository
2. Open the project in VS Code or IntelliJ IDEA
3. Compile the Java files
4. Run `Main.java`

## Example

A user can create a student, create a course, enroll the student in the course, assign a grade, and display the final result.
